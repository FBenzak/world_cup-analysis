# Copa do Mundo Dashboard / World Cup Dashboard

Projeto de análise de dados das Copas do Mundo utilizando Python e Streamlit.

Data analysis project of FIFA World Cup matches using Python and Streamlit.

---

## Funcionalidades / Features

### Português
- Análise histórica de seleções
- Estatísticas completas (vitórias, derrotas, empates, gols marcados e sofridos)
- Comparação entre países
- Score baseado em performance histórica e recência
- Dashboard interativo com Streamlit

### English
- Historical analysis of national teams
- Complete statistics (wins, losses, draws, goals scored and conceded)
- Head-to-head country comparison
- Performance score based on historical results and recency
- Interactive dashboard built with Streamlit

---

## Tecnologias / Technologies

- Python
- Pandas
- NumPy
- Streamlit
- Scikit-learn
- Plotly
- Statsmodels

---

## Dataset

### Português
Os dados foram combinados de diferentes edições da Copa do Mundo (1950–2022), incluindo partidas históricas e recentes.

### English
The dataset combines multiple FIFA World Cup editions (1950–2022), including historical and recent matches.
